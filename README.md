# bfg

`bfg` is a Brainfuck interpreter in Go.

# Usage

```bash
bfg <path-to-file>
```

# Dependencies

- Go 1.20

## Feature support

- Implements `>`, `<`, `+`, `-`, `.`, `,`, `[` and `]`.

## Contributing

You can fork this repository and run the code, make your changes to your repository and send a Pull Request.
